### 2. **Environment Detection Refactoring**
```php

    (isset($_ENV['APP_ENV']) && $_ENV['APP_ENV'] === 'testing') ||
    (isset($_SERVER['APP_ENV']) && $_SERVER['APP_ENV'] === 'testing') ||
    defined('PHPUNIT_RUNNING')

}
```
